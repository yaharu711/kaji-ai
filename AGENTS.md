# 開発時のチェックリスト

このリポジトリで開発を進める際は、以下のステップを必ず実行してから PR を作成してください。

## 1. コード整形・Lint

- VS Code を使う場合は保存時に Prettier が走る設定を入れています。
- 手動で整形したい場合は以下を実行してください。

```bash
npm run format
```

- ESLint で静的解析を実行します。エラーが出た場合は修正してから再実行してください。

```bash
npm run lint
```

- 自動修正可能なものは以下で修正できます。

```bash
npm run lint:fix
```

## 2. テスト

バックエンドの API テストを必ず通してください。

```bash
npm run test --workspace apps/backend
```

## 3. Docker 環境での確認（必要に応じて）

ローカルで動作を確認したい場合は以下でバックエンドを起動できます。

```bash
docker compose up backend
```

---

CI でも `lint` と `test` が実行されます。ローカルで同じコマンドを事前に走らせ、エラーを取り除いてからプッシュするようにしてください。分からない点があれば気軽に相談してください。よろしくお願いします！
